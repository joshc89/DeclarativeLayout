DLViewController

public class DLViewController: UIViewController

Convenience view controller subclass for creating a view controller with a Layout.

  • layout

    The UI for this view controller.

    public let layout:Layout
  • init(layout:insets:toMargins:backgroundColor:)

    Default initialiser.

    Declaration

    Swift

    public init(layout: Layout, insets: UIEdgeInsets = UIEdgeInsetsZero, toMargins:Bool = true, backgroundColor: UIColor = UIColor.whiteColor())

    Parameters

    layout

    The UI for this view controller. Set as layout.
    insets

    The inset of layout from the view controller’s view’s edge or layoutMargins depending on toMargins. Default value is zero.
    toMargins

    Flag for whether insets should be taken from the edge of the view or the layoutMargins.
    backgroundColor. The background color for the view. Default value is white.
  • init(coder:)

    Required initialiser. This will fail as this class is designed to work simply with a Layout object, rather than implementing layout code in a Storyboard.
